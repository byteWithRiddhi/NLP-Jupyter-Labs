# NLP-Jupyter-Labs

A collection of NLP practical programs implemented using Python, NLTK, spaCy, and Scikit-learn using Google Colab and Jupyter Notebook.

## Practicals Included
- Tokenization using NLTK and spaCy
- Bag of Words (BoW)
- TF-IDF Representation
- Text Preprocessing
- NLP Experiments in Python

## Technologies Used
- Python
- Google Colab
- Jupyter Notebook
- NLTK
- spaCy
- Scikit-learn
